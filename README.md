## node-editor

基于imnodes<https://github.com/Nelarius/imnodes>的流程图编辑工具。

经修改后，可以用于神经网络，或者某些特殊流程图的编辑。

目标是老妪能训。

注意除了main.cpp和color_node_editor.cpp与imnodes给出的范例不同之外，其余文件均未改动。

本工程只给出了一个简单神经网络格式的编辑和修改，如需更多格式支持可能要自己添加。
